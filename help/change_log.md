# Change Log (Previous Version Snapshot)

## v0.1.x Baseline

- Added compact two-column layout for `Tools` and `AI Features`.
- Added AI shortcuts: `Auto Sketch`, `Auto Color`, `Auto Optimize`, `Auto Resolution`.
- Added local super-resolution style selection (`General` / `Illustration`).
- Added explicit `Liquify` options panel with apply/cancel workflow.
- Added pressure curve editor integration and brush stabilize slider.
- Improved right panel scrolling for dense property content.
- Added broad shortcut support for copy/cut/paste and global undo/redo.
- Implemented redo shortcut with `Ctrl+Shift+Z`.
- Added local model path management in `Settings -> Local Models`.
- Continued PSD import/export compatibility fixes and history model improvements.
